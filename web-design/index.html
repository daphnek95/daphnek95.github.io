Shelter Dash

Shelter Dash is a Frogger style game created in Processing and then translated to javascript with P5JS. Once the game was designed, I coded a webpage for the game to lie in. The premise of the game is that you are a volunteer at an animal shelter on national Animal Adoption Awareness Day! Your boss for the day is a little scatter brained from all the commotion going on at the shelter and it's up to you to save the day.

Tetris Draw

This drawing program was designed to look and feel more geometric and digital as opposed to natural and organic like drawing normally is. Players use the arrow keys to move their brush, WASD keys to change colors, shift to erase, and the spacebar to clear the canvas. The program itself was created in processing and then translated using P5JS.
