Solar System - Interactive Water Projection

As a final project for Software Design II, we had to create an interactive program. I decided to create an interactive water projection where it switches between two different states. The first allows you to draw with a star in space. The second allows you to move around and interact with the plantes of the solar system.

Wearable Technology - The CuddlePants

For a final project in Expanded Media at ISU, we had to work in pairs to create wearable technology. My friend, Cassidy, and I decided to create pants that would cuddle a friend or significant other for you. A mechanical hand would be triggered when a head lands on the pillow. It would then stroke the person's head.

Tetris Drawing - Learning processing

This was the first program I learned how to code in my processing class at Illinois State University. The goal for the assignment was to create a drawing program where the process to draw was unique. I wanted to explore a very digital approach and followed a little tetris inspiration by drawing with computer keys rather than your mouse.
