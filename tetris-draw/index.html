Tetris draw was an exploration in using Processing to design a drawing program. Rather than making an organic or natural drawing program, I wanted to play into the digital aspect of the program and create a grided, pixelated program. Try out the program here.

The first step for this project was to build and test the program using Processing. Once that was done, it was translated to Javascript and hosted within an HTML file to live on the internet.
